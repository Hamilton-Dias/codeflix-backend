# Codeflix backend

Application written in python 3 for Full Cycle 3.0 course.
- DDD
- Clean archtecture
- Python 3
- Python venv
- Django
- RabbitMQ
- Domain Events
- Unit and Integration tests
- E2E Tests


To run the program:
<pre>
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
</pre>
